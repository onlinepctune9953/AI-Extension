Help & Support

Getting Started

AI Affiliate Pro helps you create high-quality affiliate content for Amazon products. Here's how to get started:

  1. Visit any Amazon product page
  2. Click the "Start Article Wizard" button
  3. Follow the steps to generate your content
  4. Download or publish to WordPress

Setting Up Your Account

Before creating your first article, configure these essential settings:

  1. Enter your Amazon Associate ID in the main popup or Settings panel
  2. Add your AI API keys in Settings ‚Üí AI Settings
  3. Configure your default article preferences in Settings ‚Üí General
  4. If you plan to publish to WordPress, add your site details in Settings ‚Üí WordPress
  5. Review advanced options in Settings ‚Üí Advanced

Amazon Associate ID

Your Amazon Associate ID is required to generate affiliate links. You can find it in your Amazon Associates account (e.g., yourname-20).

Without a valid Associate ID, the extension will still work, but generated links won't include your affiliate code.

AI API Keys

AI Affiliate Pro requires API keys to generate content. You can set these up in Settings ‚Üí AI Settings.

  • OpenAI API Key: Get your key from https://platform.openai.com/api-keys
  • Anthropic API Key: Get your key from https://console.anthropic.com/keys

At least one API key is required for the extension to function properly.

Creating Articles

AI Affiliate Pro offers multiple article types to suit your needs:

  • Product Review: In-depth analysis of a single product
  • Comparison: Compare multiple products side-by-side
  • Buying Guide: Comprehensive guide for a product category
  • How-to/Tutorial: Step-by-step instructions related to a product
  • Top 10 List: Curated list of best products in a category
  • Problem/Solution: Address specific problems solved by a product

Article Editor

The article editor provides a full set of formatting tools:

  • Text formatting (bold, italic, underline)
  • Headings and subheadings
  • Lists (ordered and unordered)
  • Tables for product comparisons
  • Image insertion and management
  • Affiliate link insertion

You can resize the editor window by dragging the bottom-right corner to adjust to your preferred size.

Downloading Articles

To download your article:

  1. Click the "Download" button in the editor
  2. Select your preferred format (HTML, Markdown, Word, PDF)
  3. Click "Download" to save the file to your computer

What is Competitor Research?

The competitor research feature analyzes top-ranking content for your target keywords to help you create more competitive articles. This helps you understand what content is currently ranking well and what elements you should include in your own content to outrank competitors.

Using Competitor Research

To use the competitor research feature:

  1. Visit an Amazon product page
  2. Click the "Competitor Research" button in the popup
  3. The extension will analyze top-ranking content related to the product
  4. Review the research report to understand what makes top content successful
  5. Use these insights when creating your own article

Research Depth Options

You can configure the depth of competitor research in Settings ‚Üí AI Settings:

  • Basic: Analyzes top 3 competitors (fastest)
  • Standard: Analyzes top 5 competitors (recommended)
  • Deep: Analyzes top 10 competitors (most comprehensive)

Deeper research provides more insights but takes longer to complete.

Research Report Elements

The competitor research report includes:

  • Common keywords and phrases used by top-ranking content
  • Average word count and content structure
  • Types of headings and subheadings used
  • Common questions answered in the content
  • Types of media included (images, videos, tables)
  • Recommendations for outranking competitors

WordPress Integration

To publish directly to WordPress:

  1. Go to Settings ‚Üí WordPress
  2. Click "Add New Site"
  3. Enter your site details
  4. Create an application password in WordPress (Users ‚Üí Profile ‚Üí Application Passwords)
  5. Test the connection and save

Publishing to WordPress

To publish your article to WordPress:

  1. Click the "Publish" button in the editor
  2. Select your WordPress site from the dropdown
  3. Choose a category
  4. Select publishing options (publish immediately, schedule, or save as draft)
  5. Click "Publish" to send the article to your site

WordPress Troubleshooting

If you encounter issues with WordPress publishing:

  • Verify your site URL is correct and includes http:// or https://
  • Ensure your application password is entered correctly
  • Check that your WordPress site has the REST API enabled
  • Verify your user has appropriate publishing permissions
  • Try regenerating a new application password

Common Issues

Extension Not Detecting Amazon Products

If the extension isn't detecting Amazon product pages:

  • Make sure you're on a valid Amazon product page
  • Try refreshing the page
  • Check if you're using a supported Amazon domain
  • Disable other extensions that might interfere with page content

AI Content Generation Issues

If you're experiencing problems with AI content generation:

  • Verify your API keys are entered correctly in Settings ‚Üí AI Settings
  • Check that you have sufficient credits/quota with your AI provider
  • Try switching to a different AI model
  • Reduce the word count or complexity of your request

Editor Issues

If you're having problems with the article editor:

  • Try refreshing the page
  • Clear your browser cache
  • Make sure your browser is up to date
  • Try a different browser

Frequently Asked Questions

Do I need to purchase API credits separately?

Yes, you need to create accounts with OpenAI and/or Anthropic and purchase credits directly from them. AI Affiliate Pro uses your API keys to generate content but does not include any API credits.

How much do API credits cost?

API pricing varies by provider. OpenAI charges approximately $0.01-0.03 per 1,000 tokens (roughly 750 words) depending on the model. Anthropic has similar pricing. A typical 2,000-word article might cost $0.05-0.15 in API fees.

Is my API key secure?

Yes, your API keys are stored locally in your browser's secure storage and are never sent to our servers. They are only used to make direct API calls to the AI providers.

Which Amazon domains are supported?

AI Affiliate Pro supports all major Amazon domains including .com, .co.uk, .ca, .com.au, .de, .fr, .it, .es, .co.jp, .in, and more.

Can I use the extension on multiple computers?

Yes, you can install the extension on multiple computers. However, you'll need to set up your API keys and preferences on each device.

How do I update my Amazon Associate ID?

You can update your Amazon Associate ID in the main popup or in Settings ‚Üí General.

Does the extension work with Amazon Prime Day or special promotions?

Yes, the extension works with all Amazon product pages, including special promotions and deals.

Can I edit the generated content?

Yes, all generated content is fully editable in the article editor before downloading or publishing.
